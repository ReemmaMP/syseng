Spot Markets (Tourist App)

Systems Engineering 2016/17 Group 7

Spot Markets is a collaborative project between Microsoft, Kent County Council and UCL. It promotes business in small local communities, such as in Kent. By applying Machine Learning to social media and purchase data, the project is able to make purchase recommendations to users via a Spot Markets app. Our project is a tourism extension for Spot Markets to allow users to see local Points of Interest, as well as the weather where they are.

Our app is cross-platform for iOS and Android and is written using C# and Xamarin. The backend is a Django and PostgreSQL web application and API hosted on Microsoft Azure. The recommendation engine it is built for is written in Python, connected to Azure Machine Learning and also hosted on Azure.

Feature Overview

The requirements given to us by the clients include both features we should include along with methodologies and technologies they prefer. The key features are listed here, and full details are on the requirements page.
  • In-app map to show the user where they are and what landmarks are around
  • Integration with a backend API, and preferably the Spot Markets Recommendation Engine
  • Database of landmarks & attractions built into the backend
  • Push notification support
  • Web interface to allow the council to add and remove attractions
  • Weather checking functionality
